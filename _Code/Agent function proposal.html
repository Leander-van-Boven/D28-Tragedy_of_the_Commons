Take, for some simulation SSS, the set of all epochs TST_STS​ and the set of all agents ASA_SAS​. We want to define a function fff, such that:

f:TS×AS↦[0,1]f : T_S \times A_S \mapsto [0,1] f:TS​×AS​↦[0,1]

The maximum amount of energy an agent will take is a pre-defined parameter ϵmax\epsilon_{max}ϵmax​. However, we need to make sure this amount of energy is available in the current epoch. We denote this value as ϵtS\epsilon_{t_S}ϵtS​​.
